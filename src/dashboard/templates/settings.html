Setup instructions
  1. Go to Google Cloud Console → APIs → Enable Gmail API
  2. Create OAuth 2.0 credentials (Desktop app type)
  3. Download the Client ID and Client Secret
  4. Use the OAuth playground to get a Refresh Token with gmail.send and gmail.readonly scopes
  5. Paste all three values below
